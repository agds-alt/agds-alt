## üé® Format Variations

### **For PDF/Word Export:**
- Keep simple formatting (no emojis)
- Use standard fonts (Arial, Calibri, Helvetica)
- Single column layout
- Clear section headers
- Consistent spacing

### **For Plain Text (Email/Forms):**
- Remove all markdown formatting
- Use CAPS for headers
- Use --- for separators
- Simple bullet points with * or -

### **For LinkedIn:**
- Add emojis for visual appeal
- Shorter paragraphs
- More conversational tone
- Include "Open to work" badge
